# Job Descriptions

Drop .docx files here. Each file = one job listing on the Careers page.

## File naming convention

`JD_[Role_Name].docx` — e.g., `JD_Licensed_Therapist_Geriatric_OnSite.docx`

## How it works

1. Igor uploads .docx files to this folder via GitHub
2. CC reads each file with `extract-text`
3. CC adds/updates the job listing on the Careers page
4. Job rows match the existing design pattern from design-source/pages/Careers.html
5. Each job's "Apply" and "Learn More" buttons trigger the existing job application modal

## Required sections in each JD

- Title (used as the job row heading)
- Location
- Employment Type
- About Us (standard — can be skipped, BYT boilerplate used)
- About the Role
- Responsibilities
- Requirements
- Why Join Us
